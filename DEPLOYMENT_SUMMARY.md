## 🚀 **Quick Start for Developers**

```bash
# Clone the repository
git clone https://github.com/jitenkr2030/Evalyzer.git
cd Evalyzer

# Install dependencies
bun install

# Set up environment
cp .env.example .env
# Edit .env with your configuration

# Setup database
bun run db:push
bun run db:generate

# Start development
bun run dev
```
