# Contributing to Evalyzer

Thank you for your interest in contributing to Evalyzer! This document provides guidelines and information for contributors.

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ and npm/yarn/bun
- Git
- Basic knowledge of TypeScript, React, and Next.js

### Setup

1. Fork the repository
2. Clone your fork locally
3. Install dependencies: `bun install`
4. Set up environment variables
5. Run database setup: `bun run db:push`
6. Start development server: `bun run dev`

## 📋 Development Guidelines

### Code Style

- Use TypeScript for all new code
- Follow ESLint configuration
- Use Prettier for formatting
- Write meaningful commit messages

### Component Guidelines

- Use shadcn/ui components when possible
- Follow existing component patterns
- Add proper TypeScript types
- Include accessibility features

### API Development

- Follow RESTful conventions
- Include proper error handling
- Add input validation
- Write API documentation

## 🐛 Bug Reports

When reporting bugs, please include:

- Environment details
- Steps to reproduce
- Expected vs actual behavior
- Screenshots if applicable

## 💡 Feature Requests

- Describe the use case
- Explain proposed solution
- Consider alternatives
- Include mockups if helpful

## 📝 Pull Request Process

1. Create feature branch from `master`
2. Make changes following guidelines
3. Test thoroughly
4. Update documentation
5. Submit pull request with clear description

## 🧪 Testing

- Write tests for new features
- Ensure existing tests pass
- Test across different browsers
- Check mobile responsiveness

## 📖 Documentation

- Update README for new features
- Add code comments for complex logic
- Document API endpoints
- Include examples in documentation

## 🤝 Community

- Be respectful and inclusive
- Help others in discussions
- Share knowledge and experiences
- Follow code of conduct

## 📞 Getting Help

- Create GitHub issue for bugs
- Start discussion for questions
- Check existing documentation
- Join community channels

Thank you for contributing to Evalyzer! 🎉
